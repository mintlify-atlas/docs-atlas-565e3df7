## What is this project?

Developer Portfolios is a GitHub repository started by [Emma Bostian](https://github.com/emmabostian). It hosts a single alphabetically-sorted `README.md` file containing links to developer portfolio websites, each optionally annotated with the owner's title or expertise. The list has grown to over 1,600 entries contributed by developers from around the world.

<Steps>
  <Step title="Find inspiration">
    Browse the [full portfolio list](https://github.com/emmabostian/developer-portfolios) on GitHub, or jump to a letter section using the alphabetical index at the top of the README.
  </Step>
  <Step title="Add your portfolio">
    Fork the repository, add your entry in alphabetical order using the correct Markdown format, then open a pull request.
  </Step>
  <Step title="Use the data feed">
    Fetch `feed.json` to get all entries as structured JSON — useful for building portfolio discovery tools, search pages, or statistics dashboards.
  </Step>
</Steps>
